# Lernora 🎓

A gamified learning dashboard built using HTML, CSS, and Vanilla JavaScript.

## 🚀 Features
- Dynamic lesson progression
- XP & Level system
- Achievement unlocking system
- Daily goal progress tracking
- Recent activity tracking
- LocalStorage persistence
- Responsive design
- Reset progress functionality

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript (ES Modules)
- LocalStorage API

## 📦 Architecture
- data.js – initial state
- storage.js – localStorage handling
- ui.js – DOM rendering logic
- main.js – core state logic

## 💡 Future Improvements
- Toast notifications instead of alerts
- Smooth XP animation
- Backend integration
- Dark mode

---

Built as a portfolio project to demonstrate frontend fundamentals and state management.
